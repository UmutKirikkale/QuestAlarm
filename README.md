# QuestAlarm

Retro 8-bit RPG alarm app for Android. Wake up by shaking your phone to defeat the morning monster.

## Telefona yükleme (en kolay)

Bilgisayarda Flutter kurmadan APK almak için: **[KURULUM.md](KURULUM.md)** dosyasına bakın (GitHub Actions ile otomatik APK).

## Geliştirici

```bash
flutter pub get
flutter run
```
